What to Do?

  • Use the Open3D library in Python.
  • You can take any 3D object (formats .ply, .obj, .stl, .off, etc.).
    • Do not use the same models - each student must have a unique 3D model.
    • You can find free models on websites such as https://free3d.com/3d-models and others.
  • Perform each operation step by step, visualizing the results in one script. After each stage, a new window (draw_geometries) should open.
  • After each step, you must explain what you understood and print to the console:
    • number of vertices (points),
    • number of triangles (if applicable),
    • number of voxels (if applicable),

Tasks:

The work is accepted only if all stages are completed. Missing even one step — 0 points. Full and correct completion of all 7 steps — 100 points.

1. Loading and Visualization

  • Load any 3D file (ply, obj, stl, etc.).
  • Display the original model.
  • Print to the console:
    • number of vertices,
    • number of triangles,
    • presence of color and normals.

2. Conversion to Point Cloud

  • Convert the model into a point cloud using o3d.io.read_point_cloud().
  • Display the cloud.
  • Print to the console:
    • number of vertices,
    • presence of color (if applicable).

3. Surface Reconstruction from Point Cloud

  • Reconstruct a mesh using create_from_point_cloud_poisson().
  • Remove artifacts using the .crop() method with a bounding box.
  • Display the reconstructed object.
  • Print to the console:
    • number of vertices,
    • number of triangles,
    • presence of color.

4. Voxelization

  • Convert the point cloud into a voxel grid (create_from_point_cloud).
  • Adjust the voxel size (voxel_size = 0.05 or another).
  • Display the voxels.
  • Print to the console:
    • number of vertices,
    • presence of color.

5. Adding a Plane

  • Create any plane in the scene — horizontal, vertical, or inclined.
  • Place it next to the object.
  • Display them together.

6. Surface Clipping

  • Implement a simple intersection: remove all points that lie on the “right side” of the plane.
  • Show the model after clipping.
  • Print to the console:
    • number of remaining vertices,
    • number of triangles,
    • presence of color and normals.

7. Working with Color and Extremes

  • Remove the original colors and apply your own gradient (along the X, Y, or Z axis).
  • Find the extreme points (minimum and maximum along the chosen axis).
  • Highlight them (sphere, cube, or wireframe).
  • Print to the console:
    • coordinates of the extrema.

Что делать?

  • Используется библиотека Open3D на Python.
  • Можно взять любой 3D-объект (форматы .ply, .obj, .stl, .off и др.).
    • Не берите одинаковые модели — у каждого студента должна быть уникальная 3D-модель.
    • Можно искать бесплатные модели на сайтах вроде https://free3d.com/3d-models и другие.
  • Каждую операцию выполняйте поэтапно, визуализируя результаты одним скриптом.
    После каждого этапа должно открываться новое окно (draw_geometries).
  • После каждого шага обязательно расскажите что вы поняли и выводите в консоль:
    • количество вершин (points),
    • количество треугольников (если применимо),
    • количество вокселей (если применимо),
    • наличие цвета / нормалей / пересечений.

Задания:

Работа принимается только при выполнении всех этапов. Пропуск хотя бы одного пункта — 0 баллов. Полное и корректное выполнение всех 7 шагов — 100 баллов.

1. Загрузка и визуализация

  • Загрузите любой 3D-файл (ply, obj, stl и т.д.).
  • Отобразите исходную модель.
  • Выведите в консоль:
    • количество вершин,
    • количество треугольников,
    • наличие цвета и нормалей.

2. Преобразование в облако точек

  • Преобразуйте модель в point cloud через o3d.io.read_point_cloud()
  • Отобразите облако.
  • Выведите в консоль:
    • количество вершин,
    • наличие цвета (если приемлимо).

3. Реконструкция поверхности из облака точек

  • Восстановите mesh через create_from_point_cloud_poisson().
  • Удалите артефакты методом .crop() по bounding box.
  • Отобразите реконструированный объект.
  • Выведите в консоль:
    • количество вершин,
    • количество треугольников,
    • наличие цвета.

4. Вокселизация

  • Превратите облако точек в воксельную сетку (create_from_point_cloud).
  • Подберите размер вокселя (voxel_size = 0.05 или другой).
  • Отобразите воксели.
  • Выведите в консоль:
    • количество вершин,
    • наличие цвета.

5. Добавление плоскости

  • Создайте любую плоскость (plane) в сцене — горизонтальную, вертикальную или наклонную.
  • Расположите рядом с объектом.
  • Отобразите их вместе.

6. Обрезка по поверхности (клиппинг)

  • Реализуйте простое пересечение: удалите все точки, которые лежат “по правую сторону” от плоскости.
  • Покажите модель после обрезки.
  • Выведите в консоль:
    • количество оставшихся вершин,
    • количество треугольников,
    • наличие цвета и нормалей.

7. Работа с цветом и экстремумами

  • Уберите исходные цвета, задайте собственный градиент (по оси X, Y или Z).
  • Найдите экстремальные точки (минимум и максимум по выбранной оси).
  • Выделите их (сфера, куб или wireframe).
  • Выведите в консоль:
    • координаты экстремумов.
